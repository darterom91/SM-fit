A Simple plugin to check the Plugin Compatibility with the Latest Stable version of WordPress

== Description ==
A Simple Plugin to Check that Your Plugin is compatible up top which version of WordPress, before performing WordPress Update. It will show you the current running plugin version, the latest release of the plugin version, and are they ready to update or not with the latest version of WordPress.

== Screenshots ==

1. The main plugin interface.

==Features==
1. Plugin Current Version Check
2. Plugin Latest Release Version Check
3. Plugin Ready to Update Or Not
4. Filter to check plugin activated or not

== Installation ==

1. Upload `plugin-compatibility-checker` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
